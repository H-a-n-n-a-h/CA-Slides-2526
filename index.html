Perceptron Algorithm 


Input: Training data { (x_i, y_i) }, x_i ∈ ℝᵈ, y_i ∈ {-1,+1}, 
	    step size η > 0
Output: Weight vector w, bias β

Initialize w ∈ ℝᵈ, β ∈ ℝ
repeat
    # Loop through each training example
    for each (x_i, y_i):
        # Check if x_i is misclassified
        if y_i * (wᵀ x_i + β) ≤ 0:
            w ← w + η * y_i * x_i
            β ← β + η * y_i

until no misclassifications

Perceptron Loss function

\begin{align*} \mathcal{L}_{\mathcal{P}}: \mathbb{R}^d \to \mathbb{R}, \quad w\mapsto - \sum_{m\in \mathcal{M}}w^\top x_m y_m \end{align*}

Counts how many datapoints are misclassified
